CTSM_PY
=======

A place to put sample workflows and tools that use ctsm model output

## Installing

First clone this repository:

```
git clone https://github.com/NCAR/ctsm_py.git
```

Then intall the utilities:

```bash
cd ctsm_py
conda activate $ENVIRONMENT_NAME
pip install -e .
```
